Campus Life

School Library

Was built in 2011

The school library has all kinds of books. We can start going to the library from 3rd class, and we can start issuing the books in the library for one week from 3rd class itself. We get library period for 30 minutes every week in a certain day for all classes starting from 3rd.



school auditorium

Was Built in 2011

The school auditorium is a big place. It is used for annual days, exhibitions, festival celebrations (before the actual festival day), award giving/announcing winners of inter-school competitions and assemblies.
